# Jairus Guirao - Personal Music Website

## Overview

This is the personal website for Jairus Guirao, a professional lead guitarist and music creator. The site showcases his musical journey, band information, music projects, skills, and provides a way for fans and collaborators to get in touch.

## Features

- **Hero Section**: Eye-catching introduction with background image and call-to-action
- **About Section**: Personal story and musical influences
- **Band Section**: Information about Nyxxed Berries band members and their journey
- **Music & Projects**: Showcase of albums, live sessions, and featured tracks
- **Gallery**: Photo gallery of performances and behind-the-scenes moments
- **Skills Section**: Display of musical expertise and proficiency levels
- **Contact Form**: Interactive form for inquiries and collaborations
- **Responsive Design**: Optimized for desktop and mobile devices
- **Lightbox Gallery**: Click-to-enlarge photo viewing experience

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Custom styling with responsive design
- **JavaScript**: Interactive features including navigation, gallery, and form validation
- **Font Awesome**: Icons for visual enhancement
- **Google Fonts**: Custom typography (Playfair Display, Inter, Space Mono)

## How to Run

1. Clone or download the project files
2. Open `index.html` in any modern web browser
3. The website will load and be fully functional locally

No server setup or dependencies are required - it's a static website.
